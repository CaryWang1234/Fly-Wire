# 图标资源文件夹

## 说明
此文件夹用于存放飞线社交平台所需的图标文件。

## 建议图标
- 应用图标 (app-icon.png, 多种尺寸)
- 飞线图标 (wire-icon.png)
- 用户图标 (user-icon.png)
- 消息图标 (message-icon.png)
- 设置图标 (settings-icon.png)

## 图标规格
- 主要图标: 64x64, 128x128, 256x256
- 小图标: 16x16, 32x32
- 格式: PNG (支持透明背景)

## 使用方式
图标可以通过CSS或HTML引用，例如：
```css
.icon {
    background-image: url('/resources/icons/icon-name.png');
}
```
